Installation Guide
Follow the simple and easy steps to have Smart PHP Poll running on your server.
Step 1: Open conf.txt, provide your database login detail
Step 2: Upload all the files into your server.
Step 3: Open setup.php in your browser. Click Start Installation button. After seeing successfull installation message, delete the file setup.php from server.
Step 4: Open admin.php in your browser. Login using default id: admin and default password: password
Step 5:Make a new poll in admin panel. You will get an ID after poll creation. Save the ID.
